# D3 Time Map for R

This is an R package that implements a time map [htmlwidget](http://htmlwidgets.org). It is based on [Times Maps: Visualizing Discrete Events Across Many Timescales](https://districtdatalabs.silvrback.com/time-maps-visualizing-discrete-events-across-many-timescales) by [Max Watson](https://github.com/maxcw/time-maps).



### Installation

To install:

```r
if (!require("devtools")) install.packages("devtools")
devtools::install_github("mnfost/d3TimeMap")
```
